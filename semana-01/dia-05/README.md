# Ficha de Cliente v1 - CampOne

Programa simples em Python para cadastro e validação de clientes.

## Funcionalidades

- Recebe dados do cliente (nome, email, telefone, status, valor do contrato)
- Valida se o nome está preenchido
- Valida se o valor do contrato é maior ou igual a R$ 500,00
- Valida se o status é "ativo"
- Exibe o resultado da validação de forma organizada

## Como executar

```bash
python ficha_cliente_v1.py
